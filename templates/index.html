4 TYPES OF ATTACKS

Attack Type Description
Denial-Of-Service(DOS)
Attack		 It is a cyber-attack in which the perpetrator seeks to make a machine
or network resource unavailable to its intended users by temporarily
or indefinitely disrupting services of a host connected to the Internet.
User to Root (U2R)
Attack		 Initially, attackers access normal user account, later gain access
to the root by exploiting the vulnerabilities of the system.
Probe Attack Have an access to entire network information before introducing
an attack.
Root to Local (R2L)
Attack		 By exploiting some of the vulnerabilities of the network,
attacker gains local access by sending packets
on a remote machine.
